# Tutorials

This directory contains step-by-step tutorials for learning various topics. Tutorials are comprehensive learning paths that take you from beginner to competent in a specific area.

## Structure

- **programming/** - Tutorials for programming languages (Python, JavaScript, etc.)
- **tools/** - Tutorials for specific tools and frameworks (Git, Docker, etc.)
- **concepts/** - Tutorials for theoretical concepts (Algorithms, Design Patterns, etc.)

## How to Use

1. Choose a tutorial that matches your learning goals
2. Follow the sections in order
3. Complete all exercises
4. Use the checkpoints to validate your understanding
5. Work through the practice project

## Creating a Tutorial

See the template in `/templates/tutorial-template.md` and refer to `/CONTRIBUTING.md` for guidelines.

## AI Assistance

These tutorials are designed to work with AI tutors. Look for "For AI Tutors" sections at the end of each tutorial for guidance on how to best assist learners.
