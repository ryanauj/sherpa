# Tools

This directory contains reusable tools, scripts, and interactive resources that can be used across different routes.

## Purpose

Tools are practical utilities that support learning:
- **Scripts**: Automation scripts for setup, testing, etc.
- **Visualizations**: Interactive diagrams or visual aids
- **Quizzes**: Interactive assessment tools
- **Executables**: Programs learners can run and experiment with

## Organization

Tools can be organized by:
- Type (scripts/, visualizations/, quizzes/)
- Technology (git/, python/, docker/)
- Purpose (setup/, testing/, practice/)

## Usage

Tools are referenced from routes, guides, and sherpas. For example:

```markdown
See `/tools/git-visualizer/` for an interactive Git visualization.
```

## Creating Tools

Tools should:
- Be well-documented with clear usage instructions
- Include examples of how to use them
- Work across different learning contexts
- Be reusable across multiple routes

See `/CONTRIBUTING.md` for guidelines on adding new tools.
